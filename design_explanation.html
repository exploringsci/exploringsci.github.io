Why does this site look so ancient?

Back to Home!

Three reasons:

  • Want my site to load quickly. Having less bloaty JS, complex animations, and excessive features means that my site loads pretty quickly! The images are still large and take a bit to load, so I'm working on making some more simple images (probably SVGs or similar) to speed up load times.
  • I don't want the site to look like the generic "flat" design trend that is seen everywhere (though it usually looks fine, I want my site to be a bit more interesting)
  • My site must scale well to all screen sizes. I don't want elements to move around and be in weird orders if someone views my site on mobile or other screen sizes.

I'm working on a more modern design right now though so look out for that!
